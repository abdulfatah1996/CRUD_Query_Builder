## About Laravel
Laravel 9 CRUD Using Query Builder    Country Part One
A practical example of adding, modifying, displaying and deleting data from the database using the Laravel platform version 9.

## Application Install
- git clone https://github.com/abdulfatah1996/CRUD_Query_Builder.git
- cd CRUD_Query_Builder
- composer install
- copy .env.example to .env
- php artisan key:generate
- create database name CRUD_Query_Builder on xampp server http://localhost/phpmyadmin/index.php?route=/server/databases.
- php artisan migrate:fresh
- php artisan db:seed
- php artisan serve
- http://localhost:8000/countries on browser
